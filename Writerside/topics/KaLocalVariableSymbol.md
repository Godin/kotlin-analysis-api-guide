# KaLocalVariableSymbol

Represents a local variable.

## Hierarchy

Inherits from [KaVariableSymbol](KaVariableSymbol.md).

## Default values

| Member              | Value                    |
|---------------------|--------------------------|
| `callableId`        | `null`                   |
| `contextReceivers`  | `emptyList()`            |
| `isExtension`       | `false`                  |
| `location`          | `KaSymbolLocation.LOCAL` |
| `receiverParameter` | `null`                   |
